# Machine Learning
Machine learning training programs



## Steps:
1.  Create Virtual Environment:
     * ``` $ virtualenv ml```
     * ``` $ source ml/bin/activate```
     
2.  Pip install required libraries:
     * ``` $ pip install numpy ```
     * ``` $ pip install scipy ```
     * ``` $ pip install sklearn ```
     
3.  Run the sample python file:
     * ``` $ python sample.py ```
